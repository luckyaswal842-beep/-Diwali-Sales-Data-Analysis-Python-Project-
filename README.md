# 🪔 Diwali Sales Data Analysis (Python Project)
## 📌 Project Overview
- This project focuses on analyzing Diwali sales data to uncover customer purchasing behavior, sales trends, and business insights using Python.
- The goal is to help businesses understand their customers better and improve sales strategies during festive seasons.

## 🎯 Objectives
- Perform data cleaning & preprocessing
- Analyze customer demographics (Age, Gender, Marital Status)
- Identify top-performing states, occupations, and product categories
-Generate insights to support data-driven decision making
## 🛠️ Tech Stack
- Python
- NumPy – numerical operations
- Pandas – data manipulation
- Matplotlib & Seaborn – data visualization
## 📂 Dataset
- Dataset used: Diwali Sales Data
- Contains information such as:
- Customer details (Age, Gender, State)
- Orders & purchase amount
- Product categories
## 🔍 Project Workflow
1. Data Cleaning
- Removed unnecessary columns (Status, unnamed1)
- Handled missing values
- Converted data types (e.g., Amount to integer)
2. Exploratory Data Analysis (EDA)
- Checked dataset structure using .info(), .describe()
- Identified null values and data distribution
3. Data Visualization & Insights
- Gender-wise purchasing behavior
- Age group analysis
- State-wise order distribution
- Marital status impact on purchases
- Occupation-wise sales trends
- Top product categories
## 📊 Key Insights
- Females contribute more to total sales compared to males
- Age group 26–35 shows highest purchasing power
- Certain states dominate total orders, indicating strong regional demand
- Married customers tend to purchase more
- Specific product categories generate maximum revenue
## 📈 Sample Visualizations
- Bar plots for sales distribution
- Count plots for demographic analysis
- Category-wise revenue comparison
